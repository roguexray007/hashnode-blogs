# Setup

1. install wtfutil from [Github link for project](https://github.com/wtfutil/wtf).
    
2. ```bash
      cd ~/.config/wtf
      touch todo.yml
      touch notes.md
    ```
    
3. update config.yml with below config
    

```yaml
wtf:
  colors:
    border:
      focusable: darkslateblue
      focused: orange

      position:

        left: 2
        height: 1
        width: 1
      refreshInterval: 150

        height: 1
